ClassList

La propiedad classList en JavaScript es una forma conveniente de acceder y manipular las clases CSS de un elemento HTML. Proporciona métodos para agregar, eliminar y alternar clases, así como para verificar si un elemento tiene una clase específica. Esto facilita la manipulación de las clases sin tener que trabajar directamente con la cadena de texto de la propiedad className.

Métodos Comunes de classList

  • add(className): Agrega una o más clases al elemento.
  • remove(className): Elimina una o más clases del elemento.
  • toggle(className): Agrega la clase si no está presente, o la elimina si ya está.
  • contains(className): Devuelve true si el elemento tiene la clase especificada, de lo contrario, devuelve false.
  • replace(oldClass, newClass): Reemplaza una clase existente por otra.
