Propiedad Style

La propiedad style en JavaScript se utiliza para acceder y manipular los estilos CSS de un elemento HTML. Cada elemento del DOM tiene una propiedad style que es un objeto que representa los estilos en línea aplicados a ese elemento. Puedes usar esta propiedad para cambiar estilos como el color, el tamaño de fuente, el margen, el padding, y muchos otros.
