Viewport y MediaQueries

LEl viewport es el área visible de una página web en un dispositivo. En dispositivos móviles, el viewport es generalmente más pequeño que en computadoras de escritorio. Para controlar cómo se muestra una página en diferentes dispositivos, se utiliza la metaetiqueta en el del documento HTML.

Media Queries

Las media queries son una técnica de CSS que permite aplicar estilos específicos a diferentes tamaños de pantalla y características del dispositivo. Esto es esencial para el diseño responsivo, ya que permite que el diseño de la página se ajuste según el entorno en el que se visualiza.
